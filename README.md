# AI College Chatbot (Python)

This is an offline college chatbot built using Python.  
It answers common college-related questions using predefined knowledge.

## Features
- Admission, fees, hostel, placement queries
- Small talk handling
- Chat history saved to file
- Offline (no API required)

## Technologies
- Python
- Dictionary-based knowledge system
- File handling

## How to run
1. Install Python
2. Run:
   python ai_chatbot.py

## Example queries
- admission
- fees
- hostel
- placement
